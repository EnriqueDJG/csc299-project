CLI Task Manager Prototype


How to use:
1. Install Python 3
2. Keep files in same folder


Commands:
Add a task:
python task_manager.py add "My Task" --due 2025-12-10 --priority high


List tasks:
python task_manager.py list


Search tasks:
python task_manager.py search keyword


Tasks are stored in tasks.json in the same directory.
You can build on this by adding update, delete, or AI-based features later.
